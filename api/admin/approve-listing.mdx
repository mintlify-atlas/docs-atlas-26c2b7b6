---
title: "Approve Listing"
api: "PATCH /api/admin/listings/:id/approve"
description: "Approve a pending listing to make it publicly visible"
---

## Authentication

This endpoint requires:
- Valid JWT token (via cookie)
- Admin role (`isAdmin: true`)

## Path Parameters

<ParamField path="id" type="string" required>
  The MongoDB ObjectId of the listing to approve
</ParamField>

## Implementation Details

- Updates the listing's `isApproved` field to `true`
- Uses `findByIdAndUpdate` with `{ new: true }` to return the updated document
- Once approved, the listing becomes visible in public listing queries
- The approval action is permanent and cannot be undone via reject endpoint
