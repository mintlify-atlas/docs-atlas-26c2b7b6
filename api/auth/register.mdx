---
title: 'POST /api/auth/register'
description: 'Register a new user account'
---

## Authentication

This endpoint does not require authentication.

## Request Body

<ParamField body="username" type="string" required>
  The username for the new account. Will be converted to lowercase.
</ParamField>

<ParamField body="email" type="string" required>
  The email address for the new account. Must be a valid email format, will be converted to lowercase and trimmed.
</ParamField>

<ParamField body="phone" type="string" required>
  The phone number for the new account. Must be a valid phone number in format: +[country code][number] (7-14 digits).
</ParamField>

<ParamField body="password" type="string" required>
  The password for the new account. Must be at least 8 characters long.
</ParamField>

## Response

<ResponseField name="_id" type="string">
  The unique identifier for the newly created user.
</ResponseField>

<ResponseField name="username" type="string">
  The username of the newly created user.
</ResponseField>

<ResponseField name="email" type="string">
  The email address of the newly created user.
</ResponseField>

<ResponseField name="phone" type="string">
  The phone number of the newly created user.
</ResponseField>

<ResponseField name="isAdmin" type="boolean">
  Whether the user has admin privileges (defaults to false).
</ResponseField>

<Note>
  A JWT token is automatically set as an HTTP-only cookie named `jwt` upon successful registration.
</Note>

<RequestExample>

```bash cURL
curl -X POST https://api.boundry.com/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "johndoe",
    "email": "john.doe@example.com",
    "phone": "+12345678901",
    "password": "securePassword123"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.boundry.com/api/auth/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    username: 'johndoe',
    email: 'john.doe@example.com',
    phone: '+12345678901',
    password: 'securePassword123'
  })
});

const data = await response.json();
```

</RequestExample>

<ResponseExample>

```json Success Response (201)
{
  "_id": "60d5ec49f1b2c72b8c8e4f3a",
  "username": "johndoe",
  "email": "john.doe@example.com",
  "phone": "+12345678901",
  "isAdmin": false
}
```

```json Error - Missing Fields (500)

  "message": "Please fill all the fields"

```


</ResponseExample>
